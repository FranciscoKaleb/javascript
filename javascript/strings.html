[2] slice(start_pos, end_pos)

slice() extracts a part of a string and returns the extracted part in a new string. The method takes 2 parameters: start position, and end position (end not included).

Extracting a portion of an array

const originalArray = [1, 2, 3, 4, 5];

// Extract elements from index 1 (inclusive) to index 4 (exclusive)
const slicedArray = originalArray.slice(1, 4);
console.log(slicedArray); // Output: [2, 3, 4]

// If no end index is provided, slice will go until the end of the array
const restOfArray = originalArray.slice(2);
console.log(restOfArray); // Output: [3, 4, 5]

Creating a shallow copy of an array

const originalArray = [1, 2, 3, 4, 5];

// Create a new array with the same elements as the original
const copyArray = originalArray.slice();
console.log(copyArray); // Output: [1, 2, 3, 4, 5]

Extracting a portion of a string

const originalString = "Hello, World!";

// Extract characters from index 0 (inclusive) to index 5 (exclusive)
const slicedString = originalString.slice(0, 5);
console.log(slicedString); // Output: "Hello"

// If no end index is provided, slice will go until the end of the string
const restOfString = originalString.slice(7);
console.log(restOfString); // Output: "World!"

converting string into an array of characters

const originalString = "Hello";

// Use slice with no arguments to convert the string into an array of characters
const charArray = originalString.slice();
console.log(charArray); // Output: ["H", "e", "l", "l", "o"]

[3] substring()

substring() is similar to slice(). The difference is that start and end values less than 0 are treated as 0 in substring().

let str = "Apple, Banana, Kiwi";
let part = str.substring(7, 13);

[4] substr()

substr() is similar to slice(). The difference is that the second parameter specifies the length of the extracted part.

let str = "Apple, Banana, Kiwi";
let part = str.substr(7, 6);

[5] replace()

The replace() method replaces a specified value with another value in a string: it can accept regular expression.

let text = "Please visit Microsoft!";
let newText = text.replace("Microsoft", "W3Schools");

regular expression

let text = "Please visit Microsoft!";
let newText = text.replace(/MICROSOFT/i, "W3Schools");

[6] ReplaceAll()

In 2021, JavaScript introduced the string method replaceAll()

text = text.replaceAll("Cats","Dogs");
text = text.replaceAll("cats","dogs");

regular expression

text = text.replaceAll(/Cats/g,"Dogs");
text = text.replaceAll(/cats/g,"dogs");

[7] toUpperCase(), toLowerCase()

A string is converted to upper case with toUpperCase(): A string is converted to lower case with toLowerCase():

let text1 = "Hello World!";
let text2 = text1.toUpperCase();

toLowerCase

let text1 = "Hello World!"; // String
let text2 = text1.toLowerCase(); // text2 is text1 converted to lower

[8] concat() (not recommended to use, use '+' instead)

concat() joins two or more strings. It is recommended to use plus operator

let text1 = "Hello";
let text2 = "World";
let text3 = text1.concat(" ", text2);

The concat() method can be used instead of the plus operator. These two lines do the same:

text = "Hello" + " " + "World!";
text = "Hello".concat(" ", "World!");

[9] trim(), trimStart(), trimEnd()

The trim() method removes whitespace from both sides of a string

let text1 = " Hello World! ";
let text2 = text1.trim();

[10] padStart(), padEnd()

The padStart() method pads a string from the start.

let text = "5";
let padded = text.padStart(4,"0");

[11] split()

A string can be converted to an array with the split() method

text.split(",") // Split on commas
text.split(" ") // Split on spaces
text.split("|") // Split on pipe

[13] charCodeAt()

The charCodeAt() method returns the unicode of the character at a specified index in a string: The method returns a UTF-16 code (an integer between 0 and 65535).

let text = "HELLO WORLD";
let char = text.charCodeAt(0);
//outputs: 72
//unicode value of capital H

[14] indexOf()

The indexOf() method returns the index (position) the first occurrence of a string in a string

let text = "Please locate where 'locate' occurs!";
let index = text.indexOf("locate");
//output: 7

[15] lastIndexOf()

The lastIndexOf() method returns the index of the last occurrence of a specified text in a string

let text = "Please locate where 'locate' occurs!";
let index = text.lastIndexOf("locate");
//output: 21

[16] search()

The search() method searches a string for a string (or a regular expression) and returns the position of the matc

let text = "Please locate where 'locate' occurs!";
text.search("locate");
//output: 7

regexp

let text = "Please locate where 'locate' occurs!";
text.search(/locate/);
//output: 7

[17] match()

The match() method returns an array containing the results of matching a string against a string (or a regular expression)

let text = "The rain in SPAIN stays mainly in the plain";
text.match("ain");
//output: 1 ain

regexp

let text = "The rain in SPAIN stays mainly in the plain";
text.match(/ain/);
//output: 1 ain

regexp global

let text = "The rain in SPAIN stays mainly in the plain";
text.match(/ain/g);
//output: 3 ain ain ain

regexp global, case insensitive

let text = "The rain in SPAIN stays mainly in the plain";
text.match(/ain/gi);
//output: 4 ain AIN ain ain

[18] matchAll()

The matchAll() method returns an iterator containing the results of matching a string against a string (or a regular expression)

let text = "I love cats. Cats are very easy to love. Cats are very popular."
const iterator = text.matchAll("Cats");
//output: Cats, Cats

regexp, must have g or it will error

let text = "I love cats. Cats are very easy to love. Cats are very popular."
const iterator = text.matchAll(/Cats/g);
//output: Cats, Cats

regexp global, case insensitive

let text = "I love cats. Cats are very easy to love. Cats are very popular."
const iterator = text.matchAll(/Cats/gi);
//output: cats, Cats, Cats

[19] includes()

The includes() method returns true if a string contains a specified value. Otherwise it returns false

let text = "Hello world, welcome to the universe.";
text.includes("world");
//output: true
let text = "Hello world, welcome to the universe.";
text.includes("world", 12);
//output: false

[20] startsWith(), endsWith()

The startsWith() method returns true if a string begins with a specified value.

let text = "Hello world, welcome to the universe.";
text.startsWith("Hello");
//output: true
let text = "Hello world, welcome to the universe.";
text.startsWith("world", 6)
//output: true
//Check if the 11 first characters of a string ends with "world":
let text = "Hello world, welcome to the universe.";
text.endsWith("world", 11);
//output: true
