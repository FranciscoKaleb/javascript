JavaScript date object

By default, JavaScript will use the browser's time zone and display a date as a full text string

const d = new Date();
// d = Tue Jul 25 2023 14:17:57 GMT+0800 (Singapore Standard Time)
const d = new Date("2022-03-25");
// d = Fri Mar 25 2022 08:00:00 GMT+0800 (Singapore Standard Time)

Creating Date Objects

Date objects are created with the new Date() constructor. There are 9 ways to create a new date object:

new Date()
new Date(date string)

new Date(year,month)
new Date(year,month,day)
new Date(year,month,day,hours)
new Date(year,month,day,hours,minutes)
new Date(year,month,day,hours,minutes,seconds)
new Date(year,month,day,hours,minutes,seconds,ms)

new Date(milliseconds)

JavaScript new Date()

new Date() creates a date object with the current date and time

const d = new Date();

new Date(date string)

new Date(date string) creates a date object from the specified date string

const d = new Date("October 13, 2014 11:13:00");
const d2 = new Date("2022-03-25");

new Date(year, month, ...)

new Date(year, month, ...) creates a date object with a specified date and time. in terms of month, January is 0 and December is 11

const d = new Date(2018, 11, 24, 10, 33, 30, 0);
//Mon Dec 24 2018 10:33:30 GMT+0800 (Singapore Standard Time)

Previous Century

One and two digit years will be interpreted as 19xx

const d = new Date(99, 11, 24);
//December 24, 1999

new Date(milliseconds)

new Date(milliseconds) creates a new date object as milliseconds plus zero time

const d = new Date(100000000000);
//100000000000 milliseconds from January 01 1970 UTC is:
//Sat Mar 03 1973 17:16:40 GMT+0730 (Singapore Standard Time)

Displaying Dates

JavaScript will (by default) output dates using the toString() method. This is a string representation of the date, including the time zone. The format is specified in the ECMAScript specification

//Tue Jul 25 2023 13:13:52 GMT+0800 (Singapore Standard Time)

toDateString()

The toDateString() method converts a date to a more readable format

const d = new Date();
d.toDateString();
//Tue Jul 25 2023

toUTCString()

The toUTCString() method converts a date to a string using the UTC standard

const d = new Date();
d.toUTCString();
//Tue, 25 Jul 2023 06:38:59 GMT

toISOString()

The toISOString() method converts a date to a string using the ISO standard

const d = new Date();
d.toISOString();
//2023-07-25T06:39:37.014Z

JavaScript Date Formats

ISO Date "2015-03-25" (The International Standard)
Short Date "03/25/2015"
Long Date "Mar 25 2015" or "25 Mar 2015"

const d = new Date("2015-03-25");
//Wed Mar 25 2015 08:00:00 GMT+0800 (Singapore Standard Time)

const d = new Date("03/25/2015");
//Wed Mar 25 2015 00:00:00 GMT+0800 (Singapore Standard Time)

const d = new Date("Mar 25 2015");
//Wed Mar 25 2015 00:00:00 GMT+0800 (Singapore Standard Time)

Parsing Dates

If you have a valid date string, you can use the Date.parse() method to convert it to milliseconds. Date.parse() returns the number of milliseconds between the date and January 1, 1970

const msec = Date.parse("March 21, 2012");
//1332259200000

Set Date Methods

setDate() Set the day as a number (1-31)
setFullYear() Set the year (optionally month and day)
setHours() Set the hour (0-23)
setMilliseconds() Set the milliseconds (0-999)
setMinutes() Set the minutes (0-59)
setMonth() Set the month (0-11)
setSeconds() Set the seconds (0-59)
setTime() Set the time (milliseconds since January 1, 1970)
